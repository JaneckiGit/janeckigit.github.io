# Jak dodać README profilu na GitHub

GitHub wyświetla plik `README.md` z repozytorium o nazwie **`JaneckiGit`**
(nazwa repo = nazwa użytkownika) jako wizytówkę na górze Twojego profilu.

## Kroki (2 minuty)

1. Wejdź na <https://github.com/new>
2. **Repository name:** `JaneckiGit` — GitHub pokaże ✨ "You found a secret!"
3. Zaznacz **Public** oraz **Add a README file**, kliknij **Create repository**
4. Otwórz plik `README.md` w nowym repo → kliknij ✏️ (Edit)
5. Wklej całą zawartość pliku [`README.md`](./README.md) z tego folderu
6. Kliknij **Commit changes** — gotowe! Wizytówka pojawi się na <https://github.com/JaneckiGit>

## Co zawiera README

- Nagłówek ze zdjęciem, animowanym podpisem i badge'ami (portfolio, LinkedIn, e-mail, CV)
- Sekcję "About me" i pełny tech stack z ikonami
- Doświadczenie, certyfikaty (PSM I z linkiem do Credly) i wyróżnione projekty
- Automatyczne statystyki GitHub (odświeżają się same)
